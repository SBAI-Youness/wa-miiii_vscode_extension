# wa-MIIII

A VS Code extension that plays a sound when a terminal command fails.

## How to use it

1. Open the **wa-MIIII Engine** view in the Explorer sidebar.
2. Click **Enable Audio** to activate the sound engine.
3. You can **Disable Audio** at any time to mute the sounds.
4. Every time a terminal command fails (returns a non-zero exit code), you'll hear the sound!

## Features

- **Instant Feedback**: Know exactly when a command fails without looking.
- **Toggle Control**: Easily enable or disable the sound via the sidebar.
- **Lightweight**: Zero impact on performance.

## Requirements

- VS Code 1.93.0+
- Terminal Shell Integration (enabled by default in VS Code).
